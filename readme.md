# Repo Latihan Binar

### Latihan 1: Media Query
```
Buatlah sebuah halaman HTML yang responsive terhadap 3 jenis device:
Mobile, Tablet, dan Desktop
a. Ketika ukuran mobile: background color -> merah + tulisan -> Ini device mobile
b. Ketika ukuran tablet: background color -> biru + tulisan -> Ini device tablet
c. Ketika ukuran desktop: background color -> hijau + tulisan -> Ini device desktop
```
